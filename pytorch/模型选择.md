# 机器学习的两大门派
1. 决策树群
简单来说，它们都是由成百上千棵“决策树”组成的。第一棵树先去预测，肯定会有误差（残差）；第二棵树不预测原目标，而是专门去预测并纠正第一棵树犯的错；第三棵树再去纠正前两棵树的错……以此类推。这种“知错就改、不断迭代”的机制，就叫 Boosting（提升）。
  1. XGBoost：
     极限梯度提升 引入了二阶导数（泰勒展开）和强大的正则化。**当你不知道用什么时，首选它。它对各种类型的数据表现都很稳，防过拟合能力强。**

  2. LightGBM 轻量级梯度提升
       它首创了直方图算法（把连续数值分桶）和按叶子节点生长（Leaf-wise）的策略。 数据量极其巨大时（百万级甚至亿级行数）。**它的训练速度通常是 XGBoost 的好几倍，且极其节省内存。但容易过拟合，需要限制树深**

  3. CatBoost (类别型梯度提升)
       它的底层用的是“对称树”（每一层的切分条件都一样），这种结构让它抗噪能力极强。数据中包含**大量类别特征（Categorical Features，比如城市名、职业、型号**时。不需要你手动做独热编码（One-Hot），直接把文本列喂给它，表现往往惊人。
  4. 随机森林
     相对于迭代纠错的决策树 随机森林是**让一万个不同的人独立做同一张卷子，最后投票选出答案（这叫 Bagging）**
     
     难过拟合，但上限精度通常不如那决策树算法高

     因其精度不高 不适应于我们kaggle及其强烈的竞争 但是其稳定不容易过拟合 因此我们可以拿来做特征筛选
2. 深度学习算法
   当数据并非表格 excel 而是非结构化数据 (Unstructured Data) 时 我们必须使用深度学习算法 例如神经网络
   1. 图片
        我们最熟悉的CV环节 自然使用cnn来处理
   2. 生成/视频
        Diffusion Models 绝对的霸主 例如sora的 它的原理是把图片逐渐加噪变成纯雪花点，然后让神经网络学习如何从雪花点中“去噪”还原出新画面。
   3. 强化学习
       DQN、PPO。你现在每天用的 ChatGPT，之所以说话那么像人类且有礼貌，就是因为在训练的最后一步使用了 RLHF（基于人类反馈的强化学习），它用的底层算法就是 PPO。
   4. 文字
        说到文字 绝对离不开深度学习界的大变 天 transformer

简单描述一下 强化学习用的是试错学习而不是监督学习 

监督学习用的是标签填胃式教学 在 CV 或 NLP 中，模型预测错了，你会直接把标签（Label）拍在它脸上：“这只是一只狗，不是猫！调整权重！

试错学习故名思意 就是通过不断尝试  然后记忆

# 两大门派
1. DQN（离散学习）
每到一个新环境，它会在脑子里给所有能做的动作**“打分”**（预测未来能赚多少收益）。比如：向左走打 80 分，向右走打 -10 分。然后它永远挑分数最高的那个动作执行。
2. PPO（连续学习）
不给动作打分，而是直接输出一个**“概率分布”**。遇到某个情况，它凭直觉认为“70%的概率该往左，30%的概率该往右”。如果往左走摔跤了，它下次就会把往左的概率调低到 50%。
